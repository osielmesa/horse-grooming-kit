## Description

The Mane & Tail Pin Brush in the 8-piece kit serves the same function as in the 4-piece set: detangling and smoothing the mane and tail using metal pins on a rubber cushion pad. It has a long beechwood handle with a hanging hole and the "MS / MASIEL SPARK" logo laser-engraved on the back. The dimensions in this kit are slightly different from the 4-piece version — confirm with supplier if it is a different SKU.

## Function & Use

- Detangling and smoothing the mane and tail
- Metal ball-tipped pins glide through long hair without snagging
- Rubber cushion pad absorbs pressure to minimize hair breakage
- Long handle provides leverage for working through thick, knotted tails
- Can be used to apply detangling sprays or conditioners through the tail
